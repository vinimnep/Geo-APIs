# Cidades Brasileiras - Teste de geoAPI's

Aplicação web simples que utiliza as API's OpenWearth, OpenStreeMap e MediaWiki para obter alguns dados de cidades brasileiras.

## Tecnologias Utilizadas

- HTML
- CSS
- JavaScript
- Leaflet (biblioteca para mapas interativos)
- API do OpenWeather (para dados climáticos)
- API do OpenStreetMap (para dados de localização)
- API da Wikipedia (para informações sobre a cidade)
